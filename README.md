# 🎭 PIXORA v1.0

**AI-Powered Social Media Content & Calendar Assistant**

Türkçe çalışan, mobil öncelikli sosyal medya içerik üretim, görsel hazırlama ve takvim planlama uygulaması.

## 🚀 Status

🟡 **DEVELOPMENT IN PROGRESS** - 22 Agent Orchestration Active

## 📋 Tech Stack

- **Backend**: FastAPI + SQLAlchemy + PostgreSQL
- **Admin Panel**: React + Vite + TypeScript
- **Mobile**: Flutter + Dart
- **Cache/Queue**: Redis
- **Authentication**: JWT
- **AI**: OpenAI + Fallback Providers

## 📁 Project Structure

```
pixora/
├── backend/          # FastAPI backend
├── admin-panel/      # React admin panel
├── mobile/           # Flutter mobile app
├── docker-compose.yml
└── docs/             # Documentation
```
